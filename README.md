☕ Bhopal Cold Coffee Finder Web App
📌 Overview
The Bhopal Cold Coffee Finder is a web application built using Python and Streamlit that helps users discover the best cold coffee spots in Bhopal. It provides insights based on ratings, location, and availability, making it easy to explore top cafés in the city.

🚀 Features

🔍 Search cafés by name
📊 Filter based on ratings
🏆 Top-rated coffee spots leaderboard
🕒 Opening & closing time insights
📅 Month-wise data analysis
📍 Location-based café discovery


🛠️ Tech Stack
Python
Pandas
Streamlit
folium

📂 Dataset
The dataset contains:
Café Name
Latitude & Longitude
Type (Cafe, Cart,truck )
Ratings ⭐
distances 
Opening & Closing Time

💡 How It Works
Data is loaded using Pandas
Cleaned and processed for analysis
Interactive UI built using Streamlit
Users can explore cafés dynamically

▶️ Run Locally
# Clone the repository
git clone https://github.com/your-username/bhopal-cold-coffee-finder.git

# Navigate to project folder
cd bhopal-cold-coffee-finder

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
